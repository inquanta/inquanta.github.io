üí¨ | Commands

Inquanta uses a slash-command interface for easy access.

An argument enclosed in square brackets [ ] is required, while an argument in parentheses ( ) is optional.

A user who has Administrator permission or Manager Server permission is considered an admin.

/add [counter] [count]

Adds the specified amount to your contribution total.
Example: /add WritingMarathon 1200

/leaderboard [counter]

Displays the leaderboard for the specified counter.
Example: /leaderboard WritingMarathon

/view-contri

Shows a breakdown of your contributions.
Example: /view-contri

/reset [counter]

Resets your contribution.
Example: /reset WritingMarathon

/admin counter create [name] [unit] (description)

Creates a new counter.
Example: /admin counter create WritingMarathon words

/admin counter delete [name]

Deletes a specified counter.
Example: /admin counter delete WritingMarathon

/admin counter reset [name]

Resets all users' contributions.
Example: /admin counter reset WritingMarathon

/admin user ban [user] [counter]

Bans a user from contributing.
Example: /admin user ban @fancy3081 WritingMarathon

/admin user unban [user] [counter]

Unbans a previously banned user.
Example: /admin user unban @fancy3081 WritingMarathon

/admin user reset [user] [counter]

Resets a specific user's contribution.
Example: /admin user reset @fancy3081 WritingMarathon

/support

Shows support links.
Example: /support
