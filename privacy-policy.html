Last Updated: June 27, 2025

We value user privacy and only store what is strictly necessary for the bot to function.

1. Data We Collect

  • User IDs
  • Contribution Data, i.e., Counts submitted via commands (e.g., 250 words, 2 hours).
  • Server IDs

2. How Data Is Used

  • User IDs: Used to associate contributions and activity with individual users for tracking and leaderboard purposes.
  • Contribution Data: Used to display statistics, generate leaderboards, and provide feedback to users and server admins.
  • Server IDs: Used to separate and manage data for each server where the bot is active, and to determine which server's admins can modify/delete a counter.

3. Data Retention

  • Data is stored until explicitly deleted by server admins or reset via bot commands.
  • If the bot is removed from a server, associated data may be retained unless deletion is requested.

4. Third Parties

  • We do not sell or share your data with third parties.
  • If hosting on third-party infrastructure (e.g., Google Cloud), their policies apply at the infrastructure level.

5. User Rights

  • You may request the deletion of your data via the support server or by contacting the bot developer.
  • Server admins can reset or delete contribution data using bot commands.

6. Changes

  • This policy may be updated. Before any change occurs, 2 weeks notice will be provided on this page and via the bot's /about command. Continued use of the Bot after changes implies acceptance of the new policy.
